# Generador de Contraseñas Seguras
Este es un programa en Python que permite generar contraseñas seguras, solicitando al usuario la longitud deseada.

## Características
- Genera contraseñas aleatorias utilizando letras, números y caracteres especiales.
- Recomienda una longitud mínima de 8 caracteres.
- Maneja errores de entrada, como valores no numéricos o números negativos.

## Requisitos
- Python 3.12

## Instalación

1. Clona este repositorio:

```bash
git clone https://github.com/Nakajito/password_generator.git
```

2. Navega al directorio del proyecto:

```bash
cd password_generator
```

## Uso

1. Ejecuta el programa:

```bash
python password_generator.py
```

2. Ingresa la longitud de la contraseña cuando se te solicite.
3. Recibe una contraseña segura generada aleatoriamente.

## Ejemplo

```
Ingrese la longitud de la contraseña: 12
Contraseña generada: aP9$dGk@L3qZ
```

## Notas
- La longitud mínima recomendada es de 8 caracteres por seguridad.
- Si se ingresa una longitud menor o un valor inválido, el programa mostrará un mensaje de error.
